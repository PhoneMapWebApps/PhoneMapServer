Each submitted tasks should consist of one javascript file and multipe other files enclosed in one zip folder. When the code runs it is passed the contents or one file as the 'data' parameter.

The code is executed using LiquidCore library which is used to communicate between the code and the host app.

The code should contain two listeners: 'onStart' and 'onDelete':

LiquidCore.on('onStart', function(data) {}); LiquidCore.on('onDestroy', function(force_stop) {});

  • The onStart listener is passed one parameter, this are the contents of one data file.
  • The onDestroy listener is passed one parameter, it is a boolean value specifying whether the onDestroy was called by the system.
